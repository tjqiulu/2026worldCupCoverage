# docs/context/ — 上下文文件导航

> 5 个 context 文件共同回答："这个项目现在是什么样的？怎么动？"

## 5 个文件

| 文件 | 回答什么 | 更新频率 |
|------|----------|----------|
| [project-context.md](project-context.md) | 当前快照：active plan / blocker / 文档新鲜度 | 每次 plan 状态变化 |
| [conventions.md](conventions.md) | 命名 / 风格 / 提交规范 | 极少（除非规范变了） |
| [source-of-truth-and-precedence.md](source-of-truth-and-precedence.md) | 多个真相源冲突时听谁的 | 极少 |
| [ai-autonomy-policy.md](ai-autonomy-policy.md) | 哪些能动、哪些要问、5 级自主权 | 偶尔（新 protected area 时） |
| [codebase-map.md](codebase-map.md) | 每个文件干嘛的 / 入口在哪 | 加/删/改文件时 |

## 共同原则

- 这 5 个文件**全部基于实际项目内容写**，零占位符
- 任何跟"项目长什么样"相关的问题，先来这里查
- 改任何文件前先看 `source-of-truth-and-precedence.md` 避免越权
