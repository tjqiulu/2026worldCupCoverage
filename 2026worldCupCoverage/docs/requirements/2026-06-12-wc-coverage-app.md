# 2026 World Cup Coverage App — 需求文档

> 创建日期: 2026-06-12
> 状态: **active**（驱动整个项目）
> 关联 plan: [001-initial-skeleton.md](../plans/001-initial-skeleton.md)

## 1. 背景与目标

**2026 FIFA 世界杯**于 2026-06-11 至 2026-07-19 在美国、加拿大、墨西哥三国举办，首次扩军至 48 支球队、104 场比赛。

需要一个**桌面看板**，让用户（和朋友、家人？）在 Ubuntu 桌面上随时看到赛程进展。

## 2. 用户故事

- **作为球迷**，我想一眼看到今天的比赛和比分
- **作为球迷**，我想看完整赛程（不是只今天）
- **作为球迷**，我想点进比赛看谁进的球、什么时候进的、在哪踢的
- **作为多语言使用者**，我想在中文/英文之间切换
- **作为 Ubuntu 用户**，我希望它像一个"桌面应用"——不用手动开浏览器

## 3. 功能需求

### 3.1 必须有（MVP）

| ID | 描述 | 验收 |
|----|------|------|
| F1 | 按日期分组显示 104 场比赛 | 滚动查看所有日期 |
| F2 | 当前日期高亮 | 打开时自动定位到今天 |
| F3 | 显示参赛国国旗 + 队名 | 用 flag-icons |
| F4 | 显示比分（如果已踢完） | 0-0、2-1 等 |
| F5 | 点击比赛看详情 | 弹窗：进球人、进球时间、场馆 |
| F6 | 双语 UI（zh + en） | 顶部切换器 |
| F7 | 桌面浏览器全屏 | 启动后全屏，无边框 |
| F8 | 刷新按钮 | 拉到最新 ICS |

### 3.2 应该做（V1.1）

- F9 暗色/亮色模式（auto）
- F10 收藏关注球队
- F11 显示下一场比赛倒计时
- F12 桌面通知（开赛前 N 分钟）

### 3.3 以后再说（V2+）

- F13 比赛集锦/视频链接
- F14 球员数据
- F15 移动端 PWA
- F16 多赛事支持（欧冠、欧洲杯等）

## 4. 非功能需求

- **性能**: 启动到首屏 < 2s（本地解析后）
- **可用性**: 单击即用（一条命令启动）
- **国际化**: zh-CN（默认）+ en
- **可维护性**: 文档先行（AGE 方法论）
- **离线**: 拉一次 ICS 后离线可用（缓存到 `data/matches.json`）

## 5. 约束

- 桌面 OS: Ubuntu 22.04+（用户系统）
- 浏览器: Chromium / Firefox 任一
- 不打包为 Electron（用户选 A 方案）
- 不连实时比分 API（用户选刷新模式）
- 不商用（个人项目）

## 6. 数据源

### 6.1 赛程（基础）
- **baires/fifa-cal-2026** GitHub repo
- 拉 `https://cdn.jsdelivr.net/gh/baires/fifa-cal-2026@master/calendars/en.ics`（en 版本）
- 包含 104 场比赛的小组赛 + 淘汰赛

### 6.2 进球/场馆（手写）
- 我们手维护 `data/details.json`
- 每个 match_id 存：goalscorers（人名+分钟）、venue（场馆+城市+容量）
- 比赛结束后手动更新

### 6.3 国旗
- **lipis/flag-icons** 6.6.6 via jsdelivr CDN
- `https://cdn.jsdelivr.net/gh/lipis/flag-icons@6.6.6/css/flag-icons.min.css`
- 类名格式：`fi fi-cn`（ISO 3166-1 alpha-2）

## 8. 风险

| 风险 | 概率 | 影响 | 缓解 |
|------|------|------|------|
| baires repo 维护者删库 | 低 | 中 | ICS 缓存到本地 `data/matches.json` |
| CDN 失败（flag-icons） | 低 | 中 | 降级到 emoji 国旗 |
| 浏览器全屏模式被 Ubuntu 拦截 | 中 | 中 | 备选：弹出式窗口 + F11 提示 |
| 手写 details 跟不上进度 | 高 | 低 | MVP 不强求完整，只做几场示范 |
