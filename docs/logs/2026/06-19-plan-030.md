## 根因（AGE）

### 问题 1: bracket 字体 + 宽度
- `.bracket-card` font-size 11px, padding 5px 8px
- `.bracket-labels-mirror` / `.bracket-mirror` 用 `minmax(90px, 1fr)`，min-width 950px
- 9 列 × 110px = 990px, 在较小屏幕 / 浏览器窗口下挤
- 双语 zh+en 显示 + 9 列 grid → 单卡宽度严重不足

### 问题 2: modal 没国旗
- `renderModalTeam()`（main.js:798）只检查 `team.code_iso`
- R32 卡片 `m.home.name = "1A"` 是 FIFA 字符串占位符，无 code_iso
- → 走 placeholder 分支显示 "?"

**根因同 Plan 029**：Plan 029 修了 bracket 卡片渲染（`renderMirrorCard()`），但忘了 modal 详情页也是同样的 placeholder → 应该用 `resolveBracketPlaceholder()`。

2. **复用 Plan 029 helper**：`resolveBracketPlaceholder()` 已经写过，modal 直接复用同一份代码（单一真相源）
3. **`_resolveTeamForModal` 单独 helper**：跟 `resolveTeamForMirror` 不同的 helper（modal 是 match 对象属性，不是 match.team slot）——避免一个 helper 太杂
