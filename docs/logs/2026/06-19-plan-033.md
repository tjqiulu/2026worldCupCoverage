## 修复

### Fix 1: `loadQualification()` 同步 await
**问题根因**：`loadMatches()` 流程：
```js
await loadTeams();
loadQualification();   // fire-and-forget ❌
renderBracket(allMatches);
```
bracket 渲染时 allQualification 还是 null → 全部显示 placeholder。

**修复**：
```js
await loadTeams();
try { await loadQualification(); } catch (e) { console.warn(...); }
renderBracket(allMatches);
```

### Fix 2: grid row 调整
- `grid-template-rows: repeat(8, 110px)` → `repeat(8, 100px)`
- 卡片 min-height 96px → 88px
- padding 10px→8px, font-size 14→13px
- gap 8px→6px

### Fix 3: 去掉水平滚动
- `.bracket-mirror` min-width 1100px → 0
- `.bracket-labels-mirror` min-width 1100px → 0
- minmax 130→110px（更窄也能 fit）
