## 关键决策

1. **JSON cache 而不是 Redis / DB**：项目没 Redis 依赖，JSON 文件够用（29KB）。如需重新计算直接删文件 + /api/refresh
2. **Cache 落地的时机**：仅 `/api/refresh` 触发——其他时间 read-only。避免数据不一致
3. **CSS 字号一次到位**：12→14px 不够，14→15px + 字重 700 加重视觉感知
4. **app.py 重构**：把 `_compute_and_cache_qualification` / `_api_qualification_handler` 提到模块级，create_app 内部用 `app.add_url_rule()` 注册（避免装饰器作用域问题）
