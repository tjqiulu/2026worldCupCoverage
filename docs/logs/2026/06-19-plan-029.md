## 实现内容

### Step 1: 100% 锁定 top 2 + 100% 锁定淘汰（per group）

`src/data/qualification.py` 的 `compute_per_group()` 分析每个组的出线状态：

- **locked top 2**：队 X 的当前分 > 所有其他队的 max 分（第 2 高的 max）
  - 当前实现：严格 pts 比较（不假定 GD 变化，保守）
  - 例：A 组墨西哥 6 分，其他队最高 7 分但只有 1 队 > 6 → 墨西哥锁定 1A
- **eliminated**：队 X 的 max 分 < 其他队保底分（第 2 高的当前分）
- **pending**：其他情况

### Step 2: 8 best 3rd race（跨组）

`compute_best_3rd_race()` 按 FIFA 官方 7 级优先级排序 12 个组的第 3 名：
  1. Points → 2. GD → 3. GF → 4. GA（少者优先）→ 5. Wins → 6. Draws → 7. 抽签

当前所有 12 队都是待定（小组赛才踢 1-2 轮），框架就位后会自动生效。

### Step 3: 前端 bracket placeholder resolve

- `resolveBracketPlaceholder()` 解析 `1A`/`2B`/`3X` 等 FIFA 种子占位符
- 如果对应位置 100% 锁定 → 渲染真实队 + 国旗 + 绿色边框
- 新增 CSS class `.qualified-locked`（绿色边框、粗体）
- 新增 best 3rd race 数据结构（暂未单独展示，但 API 返回完整排序）
