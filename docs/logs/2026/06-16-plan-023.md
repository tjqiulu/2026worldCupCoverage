## 遇到的关键坑

### 1. @media 被嵌套进 bracket-mirror-wrapper（CSS 语法错误）
Python 脚本移动 @media 块到 480px 块后时，`.bracket-mirror-wrapper` 缺少闭合 `}`，导致 @media 被解析为嵌套规则，**完全失效**。
**修法**: 把 @media 块移到 CSS 文件末尾（line 1326）

### 2. 首次 API 调用 17s 拖慢 Playwright 测试
`/api/matches` 首次调 worldcup26.ir 需要 ~17s enrichment。Playwright 测试 20s timeout 不够。
**修法**: Audit 脚本中预热 API（curl /api/matches）后再开浏览器

### 3. Playwright "element not visible"
Mobile viewport 小（375x667），首张 match card 在折叠下方，Playwright 报 "not visible"。
**修法**: 改用 `wait_for_selector(..., state="attached")` 而不是 `state="visible"` 的 scroll 检测
