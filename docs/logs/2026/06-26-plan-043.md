## 设计决策

### 1. Quick Start 改 3 步法

旧版走 venv 路线（`python3 -m venv .venv` + `source .venv/bin/activate` + `pip install`），但本机 `ensurepip` 没装，venv 走不通。改用**系统 Python**（Ubuntu 22.04+ 默认 3.10，apt 装 Flask/requests 即可）。如要隔离推荐 `uv venv` / `pipx`，但 README 不强求。

新 3 步法：

```bash
# 1. 装依赖
python3 -m pip install -r requirements.txt

# 2. 启服务（后台 daemon，日志 /tmp/wc_server.log）
bin/serve.sh

# 3. 浏览器
xdg-open http://127.0.0.1:8766
```

### 2. Quick Start 加 3 种部署形态

- **方式 A**：本地后端 + 浏览器（最常用，30 秒）
- **方式 B**：桌面全屏 kiosk（电视/树莓派）→ `bin/launch.py`
- **方式 C**：Render 云端 → [docs/deployment/render.md](../deployment/render.md)
