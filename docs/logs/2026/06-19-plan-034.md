## 根因

我之前改的是 `.bracket-mirror`，但**真正的肇事者是 `.bracket-mirror-wrapper`**——它有 `min-width: 950px`（Plan 005/008 早期遗留），覆盖了任何后续调整。`width: 100%` 也没设置，导致 wrapper 比 grid 容器宽 → 撑出滚动条。

卡片重叠也是 100px row 仍不够（卡片实际高度 100+px 因为 padding + 内容）。

## 修复

### Fix 1: `.bracket-mirror-wrapper` 真正释放
- `min-width: 950px` → `0`
- 加 `width: 100%` 强制 fit 父容器

### Fix 2: grid row 100→120px + gap 6→12px
- 卡片实际高度 100+px（padding 8+8 + head 20 + rows 42 + foot 16 = 94px） + gap 6px = 100+ → 实际溢出
- 120px row 给 20px 缓冲

### Fix 3: 卡片 max-height: 110px（防止 font 渲染溢出）
- 加 `max-height: 110px` 强制卡片不超过 row 高度
- padding 8→6, font-size 13→12（更紧凑）

### Fix 4: SW 升版本 v7→v8
- 强刷触发新 CSS/JS 即时生效
