## 视觉参考

**Bing sports 页面**（用户提供）作为基础 layout：
- 日期作为分组 header
- 每场比赛一行/卡片
- 左：时间 + 主场 + 国旗
- 中：vs + 比分
- 右：客场 + 国旗 + 场地

## 关键交互

### 比赛卡片
- **默认态**: 浅色背景，国家色细线
- **悬停态**: 浅阴影 + 鼠标变 pointer
- **当前比赛（live）**: 左侧红色脉冲条
- **已结束（final）**: 比分加粗
- **未开始（scheduled）**: 比分显示 "vs"

### 点击行为
- **点击卡片** → 弹窗显示详情
- **详情弹窗**:
  - 大国旗 + 大队名
  - 大比分
  - 进球列表（"🇧🇷 内马尔 23'"）
  - 场地 + 城市 + 容量
  - 开赛时间（用户本地时区）
  - "关闭" 按钮 / 点击外部关闭

### 顶部条
- 标题（点击回到今天）
- "今天" 按钮（滚动到今日）
- "刷新" 按钮（拉最新数据，loading 态）
- 语言切换（zh / EN）

## 颜色

- **主色**: 番茄红 `#E63946`（FIFA 风）
- **次色**: 海军蓝 `#1D3557`
- **强调**: 金黄 `#F1C40F`（奖杯）
- **当前比赛**: 红色脉冲 `#E63946`
- **已结束**: 中性灰 `#6C757D`
- **未开始**: 浅灰背景

## 字体

- 中文: 思源黑体 / Noto Sans CJK
- 英文: Inter / system-ui
- 比分: 等宽字体（数字对齐）

## 响应式

- **桌面（>= 1024px）**: 卡片宽度 1000px 居中
- **平板（768-1023px）**: 卡片宽度 90%
- **手机（< 768px）**: 卡片堆叠，国旗缩小

## 暗色模式

- 自动跟随系统
- 背景 `#0E1117`，文字 `#FAFAFA`
- 国旗加 filter 提亮

## 不做的事

- ❌ 不用 carousel（滚动即可）
- ❌ 不用 modal 库（手写简单弹窗）
- ❌ 不用动画（用户要"快"）
- ❌ 不用图表（无数据可图）
