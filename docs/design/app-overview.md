## 视觉参考

**Bing sports 页面**（用户提供）作为基础 layout：
- 日期作为分组 header
- 每场比赛一行/卡片
- 左：时间 + 主场 + 国旗
- 中：vs + 比分
- 右：客场 + 国旗 + 场地

## Tab 1: 赛程 Matches（默认关闭，但 widget 模式独占）

- 按日期分组所有 104 场比赛
- 今日高亮（左侧红条 + 黄色背景）
- 比赛卡点击 → 详情弹窗

## Tab 2: 对阵 Bracket（默认开启）

- 32 强 → 16 强 → 8 强 → 半决赛 → 决赛（5 列镜像布局）
- 锁定前（FIFA 占位符 `1A/2B/3A/B/C/D/F`）→ 锁定后（实队 + 国旗）
- `locked` 绿底，`favored` 琥珀底（Plan 036 引入）
- R32 占位符用 `resolveBracketPlaceholder()`（Plan 029-030）从 `/api/qualification` 解出来

## Best 3rd Top 8 面板（Plan 042）

- 位置：`</main>` 之后，独立 `<section>`，所有 tab 可见
- 12 行：1-8 名浅绿底 + ✅ 晋级，9-12 名浅灰底 + ❌ 淘汰
- 中间一条 dashed 分隔线（视觉上把晋级 / 淘汰切开）
- 顶部「数据更新于 YYYY-MM-DD HH:MM (北京时间)」从 `qualification_cache.json` 的 `generated_at` 读
- 数据源：`/api/qualification` → `best_3rd_race.rankings`
- 触发刷新：右上「🔄 刷新」按钮（自动覆盖，无需新加事件）

## 比赛详情弹窗（Plan 009-010）

- 触发：点击比赛卡（Matches tab）或 Bracket 卡片
- 内容（按 Plan 010-011 顺序）：
  1. 阶段（`小组赛 A 组` / `1/8 决赛` / `半决赛` / `决赛`）
  2. 日期 + 北京时间
  3. 主队 vs 客队（大国旗 + 中英名）
  4. 比分（已踢完）/ 倒计时（未踢）/ 实时（live）
  5. 进球列表（"🇧🇷 Neymar 23'" / "🇦🇷 Messi 45'+2'"，含点球 / 乌龙 / 伤停补时）
  6. 积分榜（仅小组赛，显示该组当前排名）
  7. 场馆（名字 + 城市 + 国家 + 容量）
- 关闭：× 按钮 / 点外部 / Esc 键

## 关键交互

### 比赛卡片
- **默认态**: 浅色背景，国家色细线
- **悬停态**: 浅阴影 + 鼠标变 pointer
- **当前比赛（live）**: 左侧红色脉冲条
- **已结束（final）**: 比分加粗
- **未开始（scheduled）**: 比分显示 "vs"
- **Plan 035 锁定态**: 绿色边框 + 浅绿底
- **Plan 036 软锁定态**: 琥珀边框 + 浅黄底

### 顶部条
- 标题（点击无操作；保留视觉锚点）
- **📲 安装**（隐藏，直到浏览器触发 `beforeinstallprompt`）— 把页面装成桌面 PWA
- **📅 今天** 滚动到当前日期（Matches tab）
- **🔄 刷新** 拉 `/api/refresh`（含 ICS + API + qualification 重算），loading 态期间禁用

### i18n

**没有语言切换按钮**——双语并列（"🇲🇽 墨西哥 / Mexico"），原因：用户群体多语言并列即可，省一个 UI 控件。

## 颜色

- **主色**: 番茄红 `#E63946`（FIFA 风）
- **次色**: 海军蓝 `#1D3557`
- **强调**: 金黄 `#F1C40F`（奖杯）
- **当前比赛**: 红色脉冲 `#E63946`
- **已结束**: 中性灰 `#6C757D`
- **未开始**: 浅灰背景
- **Top 8 晋级**: 浅绿 `#F0FDF4` + 绿 badge `#16A34A`
- **Top 8 淘汰**: 浅灰 `#F9FAFB` + 灰 badge `#E5E7EB`
- **锁定**: 浅绿 `#F0FDF4` + 边框 `#16A34A`
- **软锁定**: 浅黄 `#FFFBEB` + 边框 `#D97706`

## 字体

- 中文: 思源黑体 / Noto Sans CJK
- 英文: Inter / system-ui
- 比分: 等宽字体（数字对齐）

## 响应式

- **桌面（>= 1024px）**: 卡片宽度 1000px 居中，Top 8 面板全宽
- **平板（768-1023px）**: 卡片宽度 90%
- **手机（< 768px）**: Bracket tab 隐藏（垃圾体验），Matches 单列；Top 8 表格横向滚动

## Widget 模式

- 入口：`http://127.0.0.1:8766/?view=widget`
- 用途：桌面 / 副屏小卡片
- 紧凑布局：380x500 左右，60s 静默自动刷新
- 顶部条 + tab 都隐藏，只显示比赛卡

## 暗色模式

- v1 未做（不必要：番茄红主色在浅色下效果最好）
- 系统暗色模式下页面仍亮色，浏览器原生滚动条变暗

## 不做的事

- ❌ 不用 carousel（滚动即可）
- ❌ 不用第三方 modal 库（手写）
- ❌ 不用图表（无数据可图）
- ❌ 不用 React/Vue（零打包负担）
- ❌ 不用 i18n 切换（双语并列即可）
- ❌ 不用暗色模式（番茄红主色 + 浅色背景效果最好）
